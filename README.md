# Paperview
A newspaper like multicolumn reader mode for long form articles.

## Installation
### Bookmarklet
Visit [charsi.github.io/paperview](https://charsi.github.io/paperview/) and drag the **Paperview** link to your bookmarks bar to install.

### Extension
~WIP~

## Usage
Find a web page with a long article. Click the **Paperview** bookmarklet to reformat the page free of distractions. The article will be formatted in narrow columns like a newspaper.

## Acknowledgements
Paperview uses Mozilla's [Readability](https://github.com/mozilla/readability) library that also powers Firefox's reader view. 
The bookmarklet implementation of Paperview was initially copied and adapted from [Columnate](https://github.com/anoved/Columnate).
